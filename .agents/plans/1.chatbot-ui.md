# Module 1 — Chatbot UI

⚠️ **Medium**

## What was built
- `app.py` — Streamlit chat UI with streaming responses via Ollama SDK
- `requirements.txt` — pinned deps (streamlit, ollama, python-dotenv)
- `.env` — local config (OLLAMA_HOST, MODEL_NAME)
- `.gitignore`

## Architecture
- Ollama Python SDK (`ollama.Client`) for LLM communication
- `st.session_state.messages` for conversation history (matches Ollama message format)
- `st.write_stream()` for streaming token output
- Config via env vars — Docker/K8s ready without code changes

## Run
```
pip install -r requirements.txt
streamlit run app.py
```
