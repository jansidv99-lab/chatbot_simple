# Module 7 — LLM Observability & Evaluation (Arize Phoenix)

⚠️ **Medium**

## What was built

- Phoenix pod added to Helm chart (`helm/chatbot/templates/phoenix-deployment.yaml` + `phoenix-service.yaml`)
- `openinference-instrumentation-ollama` auto-instruments every `client.chat()` call
- Traces sent to Phoenix when `PHOENIX_ENDPOINT` env var is set (injected automatically via Helm when `phoenix.enabled: true`)
- `values.yaml` toggle: `phoenix.enabled: true` deploys Phoenix and wires up the endpoint

## Architecture

```
Streamlit Pod
    │  OTLP traces → http://phoenix:6006
    ▼
Phoenix Pod (arizephoenix/phoenix:latest)
    │  Web UI
    ▼
kubectl port-forward → http://localhost:6006
```

## Deploy

```powershell
# Upgrade chart (deploys Phoenix alongside the chatbot)
helm.exe upgrade --install chatbot helm/chatbot

# Access Phoenix UI (keep running in a separate terminal)
kubectl port-forward svc/phoenix 6006:6006
# Open http://localhost:6006
```

## Disable Phoenix (if resource-constrained)

```yaml
# helm/chatbot/values.yaml
phoenix:
  enabled: false
```

Then `helm.exe upgrade chatbot helm/chatbot`.

## Evaluation (using local Ollama as judge)

```python
import phoenix as px
from phoenix.evals import OllamaModel, HallucinationEvaluator, run_evals

client = px.Client(endpoint="http://localhost:6006")
spans = client.get_spans_dataframe()
model = OllamaModel(model="lfm2.5-thinking:latest", base_url="http://localhost:11434")
evals = run_evals(spans, evaluators=[HallucinationEvaluator(model)])
```
