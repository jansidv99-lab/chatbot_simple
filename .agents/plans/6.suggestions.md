# Module 6 — Suggested Follow-up Questions

✅ **Simple**

## What was built

- `generate_suggestions(client, model, messages)` added to `app.py` — makes a non-streaming Ollama call after each assistant response, returns 2–3 follow-up question strings
- Suggestions are displayed automatically as italic text below the last message — no buttons, no click handling
- Suggestions clear when the user sends their next message
- 3 new pytest tests covering: normal return, error silence, blank-line stripping

## Architecture

```
stream_response() completes
    │
    ▼
generate_suggestions() → separate non-streaming Ollama call
    │
    ▼
session_state.suggestions = [q1, q2, q3]
    │
    ▼ (on next rerun)
"You might want to ask:" + italic list rendered above chat input
    │
    ▼
User types next message → suggestions cleared → cycle repeats
```

## Key decisions

- **No buttons** — plain styled text only, simpler code, no rerun/state complexity
- **Silent failure** — `generate_suggestions` returns `[]` on any exception; chat always works
- **Non-streaming** — suggestions are short, no need to stream them
