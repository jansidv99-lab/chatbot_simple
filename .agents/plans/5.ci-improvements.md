## Architecture

```
pull_request → main          push → main (not bot)
      │                              │
      ▼                              ▼
  validate                       validate
  (lint + helm + pytest)         (lint + helm + pytest)
      │ (PR stops here)               │ needs: validate
                                      ▼
                               build-and-push
                               (docker + gitops update)
```

## Run tests locally

```powershell
.venv\Scripts\Activate.ps1
pytest tests/ -v
```
